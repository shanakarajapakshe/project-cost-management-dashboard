# App Icons

Place these files in this directory:

| File        | Platform | Size     |
|-------------|----------|----------|
| icon.ico    | Windows  | 256×256  |
| icon.icns   | macOS    | Auto     |
| icon.png    | Linux    | 512×512  |

## Generate from single PNG

```bash
npx electron-icon-builder --input=assets/icons/icon.png --output=assets/icons
```
